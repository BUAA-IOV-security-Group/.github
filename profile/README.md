## 📖 关于我们 | About Us

**BIOS Lab (BUAA-IoV-Security Lab)** 属于北京航空航天大学网络空间安全学院，在导师王朋成老师的带领下成立。我们不仅关注传统车联网与机器人的硬件/协议安全，更在前沿 **AI 安全** 领域深耕。从底层的 CAN 总线到上层的大模型 Agent 架构，我们致力于构建全栈式的智能系统安全评估与防御体系。

### 🔍 核心研究方向 | Research Pillars

  * 🛡️ **车联网安全 (IoV Security):** 针对 CAN 总线、车载以太网及 V2X 协议的深度模糊测试与威胁建模。

  * 🤖 **机器人安全 (Robot Security):** **ROS/ROS2** 系统的内生安全研究及物理仿真环境下的漏洞攻防。

  * ⚔️ **AI 对抗与鲁棒性 (AI Adversarial):** 研究深度学习模型在自动驾驶感知层面的对抗扰动、后门注入及防御加固。

  * 🔓 **LLM 安全与越狱 (LLM Security & Jailbreaking):** 探索大语言模型的越狱提示词攻击 (Jailbreak Prompts)、隐式注入防御及安全性对齐方案。

  * ⚡ **自动化模糊测试 (Automated Fuzzing):** 结合 AI 驱动的策略发现嵌入式固件与复杂协议栈的深层逻辑漏洞。
